## Getting Started

- Clone the repository

```bash
git clone https://github.com/localhostd3veloper/code-portfolio.git
```

- Install dependencies

```bash
bun install
```

- Run the app

```bash
bun run dev
```

- Build the app

```bash
bun run build
```
